# 潮玩咨询网站设计方案

## 项目概述
- **项目名称**: ToyPulse 潮玩脉搏
- **定位**: 专业潮玩资讯、价格追踪与收藏社区平台
- **目标用户**: 潮玩收藏家、投资者、设计师、普通爱好者

## 核心功能模块

### 1. 资讯中心 (News)
- 新品发布资讯
- 品牌动态（Bearbrick, KAWS, POP MART等）
- 设计师访谈
- 展会报道（STS, PTS等）

### 2. 价格追踪 (Market)
- 实时市场行情
- 热门单品价格走势
- 拍卖结果记录
- 价格预警功能

### 3. 百科图鉴 (Wiki)
- 潮玩品牌库
- 系列图鉴
- 设计师档案
- 鉴别指南

### 4. 社区交流 (Community)
- 玩家晒单
- 交易讨论
- 收藏展示
- 问答互助

### 5. 个人中心 (Profile)
- 我的收藏
- 关注清单
- 价格提醒
- 交易记录

## 视觉风格

### 色彩方案
- **主色**: 活力橙 (#FF6B35) - 代表潮流、活力
- **辅色**: 深灰黑 (#1A1A2E) - 稳重、专业
- **点缀**: 电光蓝 (#00D9FF) - 科技感
- **背景**: 米白/浅灰 (#F5F5F7)

### 字体
- 标题: Montserrat / 思源黑体 Bold
- 正文: Inter / 思源黑体 Regular

### 设计特点
- 大圆角卡片设计
- 渐变阴影效果
- 微交互动画
- 暗色模式支持

- 分享功能

- 价格走势图
- 基本信息
- 历史成交记录
- 同款推荐

- 筛选器（品牌/系列/年份/价格）
- 网格/列表视图切换
- 单品卡片

## 技术栈建议
- **前端**: Next.js + Tailwind CSS + Framer Motion
- **后端**: Node.js / Python + PostgreSQL
- **数据**: 爬虫 + 第三方API
- **部署**: Vercel / AWS

## 数据需求
- 潮玩基础信息（名称、品牌、系列、发售日期、发售价）
- 市场价格数据（多平台抓取）
- 资讯内容（编辑+爬虫）
- 用户数据（收藏、关注、提醒）

## 竞品参考
- StockX / Goat（价格追踪模式）
- HobbyDB（图鉴模式）
- 小红书（社区模式）
